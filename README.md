# EL-NJUXun — 南大图寻

2026 南京大学 EL 大赛交互组项目。

## 项目结构

```
njuxun/
├── frontend/             # Vue 3 前端
│   ├── src/
│   │   ├── components/   # 公共组件
│   │   ├── views/        # 页面组件
│   │   ├── router/       # 路由配置
│   │   ├── utils/        # 工具函数
│   │   └── assets/       # 静态资源
│   └── public/           # 公共资源
├── backend/              # Node.js 后端
│   ├── routes/           # 路由模块
│   ├── models/           # 数据模型
│   ├── middleware/       # 中间件
│   ├── scripts/          # 数据导入脚本
│   └── public/images/    # 校园图片
├── data/                 # 数据文件
└── docs/                 # 文档
```

## 快速开始

```bash
# 前端
cd frontend && npm install && npm run dev

# 后端
cd backend && npm install && npm run dev
```
